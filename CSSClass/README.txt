Cascading stylesheet (CSS) works in conjunction with html 

Syntax
  slector {
            property1: value1; 
            property2: value2;
          }

Selector can be an html element, a class attribute or an ID attribute
 
Example html element selector 

p {
  color: red;
}

a class atribute selector example className attribute value 
.className {
  color: red;
}
.headings{
  font-family: 'Times New Romans';
}

an id attribute example myId
#myId {
  color: blue;
} 
#topNavigation {
  float: left;
}
